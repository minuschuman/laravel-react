# Laravel-React

## Pre-requisites
- PHP 8 or higher
- Composer
- NPM
- MySQL

## Installation
installing laravel
```bash
composer create-project --prefer-dist laravel/laravel ./
```

npm install
```bash
npm install
```

install react
```bash
npm install react react-dom@latest
```

install vite
```bash
npm install @vitejs/plugin-react
npm install @vitejs/plugin-react-refresh 
```
